# vi 是Linux 最经典的文本编辑器 纯键盘编辑器,没有编译器和解释器
# vim 是vi的改进版本，一般在终端里输入 vi or vim + 后缀 即可运行代码
apt-get update
apt-get install vim


vi 文件名         打开或者新建
vi 文件名 +行数    修改哪一行代码
vi 文件名 +        继续加入代码

# 生成 .py.swp 交换文件
再次打开 会报错E325，可以删除交换文件，也可以继续编辑保存并退出

# vi:三种模式 命令模式；末行模式；编辑模式。
末行模式(:)
命令模式(Esc)
关闭文件并保存(write or quit) :wq :x
编辑模式(insert) i


一定要关闭输入法.
常用命令:
1.重复
ESC 10 i print("lkasjldkjalkdjl") ESC      十遍代码发动！
.                           重复之前的命令
N                           向上
n                           向下

2.移动和选择 
gg G 10gg :10                行
ctrl+b ctrl+f H M L         屏幕
{    }                      段落
%                           括号
mx标记  'x回到标记            标记

3.编辑操作
v选择 k向上  j向下            圈选
ctrl+v                      框选
选定行之后，I，# ,ESC         圈选注释 
x                           cut
d                           delete
dd                          删除所有选定
D                           删除到行尾

4.撤销，重复
u                           撤销                 
ctrl+r                      恢复


5.替换,查找
r                           替换当前字符(替换不好用)
R                           替换光标之后所有的字符
/str                        找到所有str
*                           光标所在单词 向上寻找
#                           光标所在单词 向下寻找

6.编辑
y                           复制框选
yy                          复制当前行
p                           粘贴

7.缩进
>>                          向右缩进
<<                          向左缩进

进入末行命令：
:%s/旧文本/新文本/g          全局替换
:%s/旧文本/新文本/g          部分替换就是选中后在输入命令
:%s/旧文本/新文本/gc         确认替换，需要确认一次才能提换,按个数替换

进入编辑模式i
I                          行首插入
a                          字符后插入
A                          行尾插入
o                          下方插入空行
O                          上方插入空行

进入分屏模式:
:e .                       打开内置文件文件浏览器，浏览目录
:n                         新建文件
:w 文件名                   另存为,阶段性的备份
:sp 文件名                  横向增加分屏
:vsp 文件名                 纵向怎加分屏
ctrl+w                     进入分屏模式
w                          切换下一个窗口
r                          互换窗口
c                          关闭当前窗口
q                          退出当前窗口 
o                          关闭其他窗口
